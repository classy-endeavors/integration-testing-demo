# Recruitment Test Output Pack

This folder contains the final written output for the recruitment task, structured for review.

## Files

- `01-end-to-end-execution.md` - End-to-end walkthrough of the test workflow using Vibe Coding heavily.
- `02-ai-vs-manual-log.md` - Section-by-section record of what AI handled, where manual intervention was needed, and why.
- `03-assessment-and-hardening.md` - Final percentage-based assessment with reasoning, plus recommendations to make the test more robust against AI-assisted completion.

## Intended Reading Order

1. `01-end-to-end-execution.md`
2. `02-ai-vs-manual-log.md`
3. `03-assessment-and-hardening.md`
